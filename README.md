# Game Clone Project Blog

A development blog documenting the creation of my game clone project.

## Local Development

To run this blog locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000/game_clone_blog/`

## Deployment

This site is automatically deployed to GitHub Pages when you push to the main branch.

Visit: `https://rdpharr.github.io/game_clone_blog/`
